# World Time - Vue 101

A Vue 101 project. Perfect for presention basic features of Vue.js

## Prerequisite

- Node
- Yarn

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
